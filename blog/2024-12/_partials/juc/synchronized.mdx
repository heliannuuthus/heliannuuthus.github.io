> `synchronized` 是 Java 基于操作系统 **Mutex Lock** 的提供的**可重入**的**非公平**锁

- 作用在不同位置

  - 锁一个对象

  ```java
  private final Object lock = new Object();

  synchronized (lock) {
    // 代码块
  }

  synchronized (this) {
    // 代码块
  }

  ```

  - 锁整个类

  ```java
  synchronized (Class.class) {
    // 代码块
  }
  ```

  - 非静态方法上声明的形式锁的是**当前对象**

  ```java
  public synchronized void method() {
    // 代码块
  }

  ```

  - 静态方法上声明的形式锁的是**类**

  ```java
  public static synchronized void method() {
    // 代码块
  }
  ```
