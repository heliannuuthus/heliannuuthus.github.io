title: 向量

---

向量是具有大小和方向的量，通常用于表示空间中的位移、力、速度等物理量。
向量在数学层面可以表示为有序数列（分量），可以是二维、三维或者更高维度的。

---

slug: matrix
title: 矩阵
description: 由一个数值（标量）按行列排列的二维数组

---

作为线性代数的基本工具，用于表示和处理线性变换、系统方程组、数据集等。

### 基本特征

- 维度：矩阵的行数和列数。
- 元素：矩阵中的数值。
- 类型：矩阵有不同的类型
  - 方阵：行数和列数相等的矩阵。
  - 零矩阵：所有元素都为 0 的矩阵。
  - 单位矩阵：对角线元素为 1，其他元素为 0 的矩阵。
  - 对称矩阵：矩阵的转置等于自身的矩阵，即 $A = A^T$。
  - 对角矩阵：非对角线元素为 0 的矩阵。
  - 上三角矩阵：对角线以下元素为 0 的矩阵。
  - 下三角矩阵：对角线以上元素为 0 的矩阵。
  - 稀疏矩阵：矩阵中大部分元素为 0 的矩阵。
  - 稠密矩阵：矩阵中大部分元素不为 0 的矩阵。

slug: tensor
title: 张量
description: 多维的数学对象，是向量和矩阵的扩展

---

> 向量作为标量的拓展，矩阵作为向量的拓展，我们需要构建更多维度（轴）的数据。通常用大写字母表示，如 $\mathbf{X，Y，Z}$

> 张量可以看作成一个多维数组，具有多个索引，可以描述多个维度的关系。其阶数属性是深度学习表达高维度数据的一种方式。

### 阶数

> 张量的阶数是指张量中每个元素的索引数量。

- 标量：0 阶张量
- 向量：1 阶张量
- 矩阵：2 阶张量
- 张量：3 阶及以上张量

#### 基本性质

- 给定具有相同形状的张量，进行一元和二元运算，结果仍然是相同形状的张量。
- 张量乘以或者加上一个标量，形状不会改变，而是每个元素都乘以或者加上这个标量。

description: 标准基向量是单位向量，用于表示坐标轴的单位向量

---

> 标准基向量是单位向量，用于表示坐标轴的单位向量。

$$
\mathbf{e}_i = [0, \cdots, 0, 1, 0, \cdots, 0]^T
$$

其中第 $i$ 个元素为 1，其他元素为 0。

:::nerd

- 向量（线性映射）是各个基方向上的分量之和
- 梯度公式中，$\nabla_x f(x)$ = ($\frac{\partial f}{\partial x_1}, \cdots, \frac{\partial f}{\partial x_n})^T$ \
  也通常用基向量表示为 $\nabla_x f(x)$ = $\sum_{i=1}^n \frac{\partial f}{\partial x_i} \mathbf{e}_i$

:::
